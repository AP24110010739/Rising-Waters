About the Project

Disaster Preparedness & Climate Action

Rising Waters focuses on addressing the growing risk of floods, which have become increasingly frequent and severe due to factors such as climate change, unpredictable weather patterns, rapid urbanization, deforestation, and inadequate drainage and water management systems. Floods can have devastating consequences, including loss of human life, displacement of communities, destruction of homes and infrastructure, agricultural damage, economic losses, and long-term environmental impacts. As populations continue to grow and weather conditions become more extreme, the need for accurate flood prediction and effective disaster preparedness has become more critical than ever.

This project leverages the power of data analysis and machine learning to develop a flood prediction system capable of identifying potential flood risks before they occur. By analyzing historical weather data, rainfall patterns, water levels, and other relevant environmental factors, the system can detect trends and generate predictions that support early warning mechanisms. These insights enable government agencies, disaster management authorities, and local communities to take proactive measures, such as issuing alerts, planning evacuations, and implementing preventive strategies.

Technology Stack

Built using Python, Flask for web routing, and Scikit-learn for modeling. Secure user credentials and prediction transaction audits are persisted via an SQLite database.

Dataset & Models

Trained on meteorological records analyzing Temperature, Humidity, Cloud Cover, and seasonal/annual precipitation. The core predictive engine is powered by an optimized XGBoost Classifier.

Audit & Accountability

Ensures transparency by logging every prediction request. The history page logs who initiated the analysis, their input parameters, the timestamp, and the model's calculated probability.

How It Works

From raw weather data to actionable flood predictions — powered by cutting-edge machine learning.

Data Collection

Ingests meteorological parameters including annual rainfall, cloud visibility, seasonal rainfall, temperature, and humidity from open datasets.

Visual Analysis

Comprehensive EDA using distribution plots, box plots, heat maps, pair plots, and descriptive statistics to understand feature patterns.

Pre-processing

Missing value imputation, IQR-based outlier treatment, feature scaling with StandardScaler, and stratified train-test splitting.

Model Training

Trains and compares Decision Tree, Random Forest, KNN, and XGBoost classifiers. XGBoost selected as best performer.

Flask Deployment

Saved XGBoost model served via a Flask API with intuitive HTML interface for real-time flood risk predictions.

Live Deployment

The completed system runs as a local web server using Flask, accessible from any device on the network. Predictions are generated in real time using saved model files.
